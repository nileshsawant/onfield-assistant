
# Help notes

Please email HPC-help@nlr.gov for assistance running VASP jobs or using the VASP modules. We can provide further support as well if special VASP compilation is needed. On Kestrel, look for the `+tpc` suffix (i.e., "third-party codes) in the module name (e.g. `module load vasp/XXXX+tpc`) for a VASP build with extra capabilities enabled such as VTST.
